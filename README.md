# Library Management System

A Java-based Library Management System developed using Core Java and MySQL.
This project helps manage books, students, and book issue/return operations efficiently.

## Features
- Add, update, delete books
- Issue and return books
- Student record management
- Book availability tracking
- Console-based user interaction

## Technologies Used
- Java (Core Java)
- MySQL
- JDBC
- Eclipse IDE

## Project Structure
- POJO classes for data models
- DAO layer for database operations
- JDBC for database connectivity

## How to Run the Project
1. Clone the repository
2. Import project into Eclipse/IntelliJ
3. Configure MySQL database
4. Update database credentials in code
5. Run the main class

## Database Tables
- books
- students
- issued_books
